◆ CAP_Open()

void CAP_Open ( uint32_t  u32InFormat,
uint32_t  u32OutFormet 
)

Open engine clock and sensor clock.

Parameters
[in]u32InFormatThe bits corresponding VSP, HSP, PCLK, INFMT, SNRTYPE, OUTFMT, PDORD and PNFMT configurations.
  • VSP should be ether CAP_PAR_VSP_LOW or CAP_PAR_VSP_HIGH
  • HSP should be ether CAP_PAR_HSP_LOW or CAP_PAR_HSP_HIGH
  • PCLK should be ether CAP_PAR_PCLKP_LOW or CAP_PAR_PCLKP_HIGH
  • INFMT should be ether CAP_PAR_INFMT_YUV422 or CAP_PAR_INFMT_RGB565
  • SNRTYPE should be ether CAP_PAR_SENTYPE_CCIR601 or CAP_PAR_SENTYPE_CCIR656
  • OUTFMT should be one of the following setting
    • CAP_PAR_OUTFMT_YUV422
    • CAP_PAR_OUTFMT_ONLY_Y
    • CAP_PAR_OUTFMT_RGB555
    • CAP_PAR_OUTFMT_RGB565
  • PDORD should be one of the following setting
    • CAP_PAR_INDATORD_YUYV
    • CAP_PAR_INDATORD_YVYU
    • CAP_PAR_INDATORD_UYVY
    • CAP_PAR_INDATORD_VYUY
    • CAP_PAR_INDATORD_RGGB
    • CAP_PAR_INDATORD_BGGR
    • CAP_PAR_INDATORD_GBRG
    • CAP_PAR_INDATORD_GRBG
  • PNFMT should be one of the following setting
    • CAP_PAR_PLNFMT_YUV422
    • CAP_PAR_PLNFMT_YUV420
[in]u32OutFormetCapture output format, should be one of following setting
  • CAP_CTL_PKTEN
  • CAP_CTL_PLNEN
Returns
None

Initialize the Image Capture Interface. Register a call back for driver internal using
