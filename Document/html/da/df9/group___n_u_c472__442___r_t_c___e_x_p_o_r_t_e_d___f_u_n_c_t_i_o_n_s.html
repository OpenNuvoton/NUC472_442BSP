uint32_t  u32DestroyEn 
)

This function is used to:

  1. Enable tamper detection function.
  2. Set tamper detect pin number.
  3. Set tamper control register, interrupt.
  4. Clear tamper status.
  5. Destroy Spare Register when tamper detected
    Parameters
    [in]u32PinNumbertamper detect pin number. [ RTC_TAMPER_0 / RTC_TAMPER_1]
    [in]u32PinConditionset tamper detection condition: 1=High level detect, 0=Low level detect
    [in]u32IntFlagEnable/Disable tamper interrupt: 1=Enable, 0=Disable
    [in]u32ClearFlagClear tamper status
    [in]u32DestroyEnEnable/Disable Destroy Spare Register when tamper detected: 1=Enable, 0=Disable
    Returns
    None

◆ RTC_SetTickPeriod()

void RTC_SetTickPeriod ( uint32_t  u32TickSelection)

The function is used to set time tick period for periodic time tick Interrupt.

Parameters
[in]u32TickSelectionIt is used to set the RTC time tick period for Periodic Time Tick Interrupt request. It consists of:
  • RTC_TICK_1_SEC : Time tick is 1 second
  • RTC_TICK_1_2_SEC : Time tick is 1/2 second
  • RTC_TICK_1_4_SEC : Time tick is 1/4 second
  • RTC_TICK_1_8_SEC : Time tick is 1/8 second
  • RTC_TICK_1_16_SEC : Time tick is 1/16 second
  • RTC_TICK_1_32_SEC : Time tick is 1/32 second
  • RTC_TICK_1_64_SEC : Time tick is 1/64 second
  • RTC_TICK_1_128_SEC : Time tick is 1/128 second
Returns
None
